# Мониторинг судов в порту (СИИ практика МТУСИ)

Мини‑проект для производственной практики: детекция судов (класс `boat`) на изображениях/видео/с веб‑камеры с помощью предобученной модели YOLO, веб‑интерфейс, история запросов, статистика и генерация отчётов (PDF/Excel).

## Быстрый старт

Команды ниже выполняй **из папки** `sii_ship_monitoring`.

1) Установи зависимости:

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

2) Запусти веб‑приложение:

```bash
python -m ship_monitoring.ui_app
```

Откроется браузерный интерфейс: загрузка изображения/видео, веб‑камера, статистика, отчёты.

## Примечания

- По умолчанию используется предобученная модель Ultralytics YOLO (COCO), класс `boat` трактуется как «судно».
- История и результаты сохраняются в папку `data/`.
