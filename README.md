# Мониторинг судов в порту

Веб-приложение для детекции судов (класс `boat`) на изображениях и видео с помощью предобученной модели YOLO. Интерфейс на Gradio: загрузка изображения/видео, веб-камера, история запросов, статистика, отчёты PDF/Excel.

## Требования

- Python 3.10+
- Зависимости из `requirements.txt`

## Установка и запуск

Из корня проекта:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m ship_monitoring
```

Интерфейс откроется по адресу http://127.0.0.1:7860.

## Структура

- `ship_monitoring/` — пакет: конфиг, модель YOLO, инференс, история, отчёты, статистика, UI (Gradio).
- `tests/` — юнит-тесты (history, reports, stats).
- `data/` — история и результаты (не в репо, см. `.gitignore`).

## Использование

- **Изображение** — загрузка файла или URL, параметры conf/iou/imgsz, кнопка «Запустить детекцию».
- **Видео** — загрузка видеофайла, шаг по кадрам, аннотированное видео и ключевой кадр.
- **Веб-камера** — снимок с браузера (WebRTC) или локальная камера/поток (OpenCV).
- **История и статистика** — сводные показатели, графики Plotly, таблица записей, сводный отчёт PDF/Excel.

История сохраняется в `data/history.json`; отчёты генерируются по запросу и доступны для скачивания в интерфейсе.
